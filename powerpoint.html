PowerPoint

Here are some tips and tricks, and some resources for using PowerPoint that I've picked up over the years. I've used it a lot and run into many frustrations, but it is a remarkably powerful piece of software.

If you are teaching, then I highly recommend that you use it in the classroom.

There is alternative presentation software, but none of it comes close to PowerPoint. The closest is Keynote on macOS and iOS, but even there PowerPoint overshadows it.

Quick Access Toolbar

Add the tools that you use the most. Especially so if they aren't in the first two or three tabs in the ribbon.

To add a tool, or sub-menu of tools, right click it and choose 'add to toolbar'. This can also be done in the settings. The order can only be changed in the settings by using the up and down buttons (not keys!)

Pen colours

You can get the normal array of colours, but Microsoft have also included metallic colours, as well as a rainbow one and even one based on imagery from the Hubble Space Telescope!

Ink Replay

You can draw using any input and PowerPoint will show a live playback of what you drew.

I've found this to be great for 'What's this?' quizzes.

Change Picture

If you want to keep the format and animations of an image, but replace the actual image, then this is the tool you need.

It can be handy if you are using an existing slide as a template but don't have placeholders in place - especially useful when doing a last minute slide!

You can also use it in picture reveal games to change the images without resizing, repositioning, etc. I recommend that you make separate slides first though, and then using this tool. This way you end up with a set of slides pre-made.

Animation Painter

Copies the animations of the first clicked object and applies them to any object clicked after.

Double click the button to be able to click multiple objects to duplicate the animations to.

Click the button or outside any object to end/cancel it.

Ctrl (Cmd) + D

You know Ctrl + C and Ctrl + V. Well this is them in one key combo! And you can hold this down to rapidly duplicate an object.

Win + V - Clipboard

Rather than going back and forth with copying and pasting, do your copying all at once and then use this keyboard shortcut to open the clipboard and insert the designed text or picture.

Add Animation and Selection Panes to Quick Access Toolbar

This will allow you to have quick access to these panes. They don't automatically open all the time and to do so from the ribbon is less efficient.

Add Align and distribute to the Quick Access Toolbar

Choose the types of align and distribute you'd like and add them for easier access. Add the whole 'Arrange' sub-menu if you need several of them.

BrightCarbon Blog

BrightCarbon are experts at PowerPoint and are the creators of the add-on I recommend the most.

This blog has great advice and nicely detailed and clear guides, including how to use visual basic (VBA).

There're some 'business bingo' nonsense posts, but that doesn't detract from the value of this blog.

How to create a random order slide show.

This specific blog post will get you a slide show in a random order.

Be aware that it requires the use of VBA and for macros to be allowed. It's also a bit of a bodge, but it works and I've yet to come across any alternative for PowerPoint.

One handy thing about it is that you can save the macro in one PowerPoint presentation and not need copy it across to other presentations you want to use it in, so long as the presentation with the macro is open.

BrightSlide by BrightCarbon

An amazing addon that can save so much time and frustration. And it's free!

THOR - The Hammer

'THOR is a PowerPoint add-in that can memorize the size and position of any shape on a slide (or the size of the slide itself), then later "hammer" any other selected shape (or shapes) to the same size and position'

Not something you may need once you have a template, but incredibly useful when creating them.

And it's free!
